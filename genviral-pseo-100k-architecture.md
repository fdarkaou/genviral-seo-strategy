## The photo.ai Model (What You're Referencing)

photo.ai has ~600k indexed pages with this structure:
- `/corporate-headshots` — "AI photos for corporate headshots"
- `/coachella` — "AI photos styled for Coachella"

- `/chef-headshots` — "AI photos for chefs"

Their actual US traffic from DataForSEO: ~21,000/month. But they're indexed for 600k pages.

**The key insight:** pSEO at 100k+ pages is NOT about each page getting high traffic. It's about aggregate long-tail capture. 600,000 pages × 3 visits/month average = 1.8M monthly visits. Each page captures a tiny slice. The sum is the business.

**What makes their pages unique:** The AI-generated photos on each page ARE the content. Each page shows different AI outputs for a different theme. Same template, different AI-generated visuals.

**For GenViral:** Your equivalent is AI-generated slideshow/video/caption examples. The AI output is the unique content per page.

## Technical Implementation

### What You Need to Build

**1. Dynamic route in Next.js**
```
/templates/[platform]/[content-type]/[niche]/page.tsx
/templates/[platform]/[content-type]/[niche]/[sub-niche]/page.tsx
```

Using `generateStaticParams()` to generate all valid combinations at build time.

**2. Content generation pipeline (run once, cache forever)**
Script that:
- Loops through all niche × platform × content-type combinations
- Calls GenViral's AI to generate 5-10 example slideshows for each combo
- Saves thumbnail images to CDN (e.g., `cdn.genviral.io/templates/tiktok-slideshow-fitness-001.jpg`)
- Saves generated captions/hooks/hashtags as JSON (5 per page)
- Each page reads from these pre-generated assets

**3. Dynamic SEO metadata**
```typescript
export function generateMetadata({ params }) {
  const { platform, contentType, niche } = params
  return {
    title: `Free ${capitalize(niche)} ${capitalize(platform)} ${capitalize(contentType)} Templates — AI-Powered | Genviral`,
    description: `Create ${niche} ${contentType}s for ${platform} in seconds. Browse AI-generated ${niche} examples and customize instantly.`,
    openGraph: { ... }
  }
}
```

**4. Auto-sitemap**
Generate `sitemap.xml` from your valid combinations list. Submit to Search Console. Done.

**5. Internal linking (critical)**
Each page links to:
- Same niche, 4 other platforms (5 links)
- Same platform, 4 other content types (4 links)
- Same platform + content type, 4 other niches (4 links)
- Parent hub pages (3 links)
= 16 internal links per page × 3,000 pages = 48,000 internal links building topical authority
